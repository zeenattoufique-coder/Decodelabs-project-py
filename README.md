# To-Do List Application

A simple To-Do List application developed in Python as part of the DecodeLabs Python Internship.

## Features
- Add tasks
- View tasks
- Delete tasks
- Easy menu-driven interface

## Technologies Used
- Python

## Project Purpose
This project demonstrates the use of Python lists, loops, conditions, and user input handling to create a basic task management application.
